# **App Name**: NutriTrack Pro

## Core Features:

- User Authentication: Secure user authentication for patients and nutritionists using Firebase Authentication.
- Food Logging: Interactive food journal for patients to log meals with descriptions, photos, and context (mood, portion size).
- Nutritionist Dashboard: Dashboard for nutritionists to manage patients, view their food logs, and provide feedback.
- Personalized Feedback: System for nutritionists to provide personalized feedback on patient's food logs, including quick responses and meal planning.
- Progress Tracking: Tools for patients to track weight, measurements, and hydration, and for nutritionists to monitor their progress.
- Meal Plan Generation: Use AI to generate meal plan suggestions, to serve as a tool to the nutritionist when constructing meal plans.  Suggestions take into account foods previously eaten and goals previously set.
- Data Export: Generate PDF reports for nutritionists summarizing patient's weekly/monthly food logs and progress. The reports will incorporate branding guidance to ensure professionalism and trustworthiness when working with patients.

## Style Guidelines:

- Primary color: Calming teal (#4DB6AC) to evoke health and wellness.
- Background color: Soft, desaturated teal (#F0F4F3) to create a clean and professional feel.
- Accent color: Warm yellow (#FFCA28) for call-to-action buttons and important notifications.
- Headline font: 'Poppins' sans-serif font for a clean, contemporary feel.
- Body font: 'PT Sans' humanist sans-serif font, for good readability in reports.
- Use minimalist, line-based icons to represent food types, moods, and activities.
- Clear, calendar-based layout for the food journal and intuitive dashboards for both patients and nutritionists.
